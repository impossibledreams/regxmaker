Folder layout:
src - source code
help - help files and licensing information
icons - icons for toolbars
javadoc - javadocs
bin - binary files

Files in the root folder:
run_gui.bat - runs the GUI utility
run_cmd.bat - runs the command line utility
build.bat - rebuilds the JARs and javadocs

NOTE: You need Java in order to run these tools and it must be in path.
